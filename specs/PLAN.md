## ID 编号约定

- 功能需求 / 任务 / 验收标准 ID **在单个 feature 内编号**，跨 feature 用 `{序号}.` 前缀区分。
- 例：`1.T-001` = 序号 1 这个 feature 的 T-001；`2.F-003` = 序号 2 的 F-003。
- **跨 feature 依赖**写全限定 ID，如 `2.T-002 依赖 1.T-001`。
